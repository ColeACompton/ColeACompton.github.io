<audio>

Defines sound content.

Syntax

  • <audio>Content Goes Here</audio>

Example

  • <audio controls>
    <source src="test.ogg" type="audio/ogg">
    <source src="test.mp3" type="audio/mpeg">
    Your browser does not support the audio tag.
    </audio>

Browser Support

  • Chrome: Version 4.0 and Up Supported
  • Internet Explorer: Version 9.0 and Up Supported
  • Firefox: Version 3.5 and Up Supported
  • Safari: Version 4.0 and Up Supported
  • Opera: Version 10.5 and Up Supported
