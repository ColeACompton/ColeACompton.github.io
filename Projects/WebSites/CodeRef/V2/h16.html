<canvas>

Used to draw graphics, on the fly, via scripting.

Syntax

  • <canvas id="ID Goes Here"></canvas>

Example

  • <canvas id="canvas"></canvas>

Browser Support

  • Chrome: Version 4.0 and Up Supported
  • Internet Explorer: Version 9.0 and Up Supported
  • Firefox: Version 2.0 and Up Supported
  • Safari: Version 3.1 and Up Supported
  • Opera: Version 9.0 and Up Supported
