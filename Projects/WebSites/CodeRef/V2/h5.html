<article>

Defines an article.

Syntax

  • <article>Content Goes Here</article>

Example

  • <article>
    <p>Article</p>
    </article>

Browser Support

  • Chrome: Version 6.0 and Up Supported
  • Internet Explorer: Version 9.0 and Up Supported
  • Firefox: Version 4.0 and Up Supported
  • Safari: Version 5.0 and Up Supported
  • Opera: Version 11.1 and Up Supported
