<area>

Defines an area inside an image-map.

Syntax

  • <area shape="Value Goes Here" coords="Coordinates Go Here" href="URL Goes Here" alt="Text Goes Here">

Example

  • <img src="images/background.jpg" width="100px" height="100px" alt="imgmap" usemap="#areamap">
    <map name="areamap">
    <area shape="rect" coords="0,0,100,100" href="https://www.w3schools.com" alt="Area">
    </map>

Browser Support

  • Chrome: Supported
  • Internet Explorer: Supported
  • Firefox: Supported
  • Safari: Supported
  • Opera: Supported
