<address>

Defines contact information for the author/owner of a document.

Syntax

  • <address>Contact Information Goes Here</address>

Example

  • <address>
    Written by <a href="mailto:cole.compton.02@gmail.com">Cole Compton</a><br>
    Visit us at:<br>
    <a href="https://www.w3schools.com">https://www.w3schools.com </a><br>
    225 Honeysuckle Way, Niceville FL 32578<br>
    USA
    </address>

Browser Support

  • Chrome: Supported
  • Internet Explorer: Supported
  • Firefox: Supported
  • Safari: Supported
  • Opera: Supported
