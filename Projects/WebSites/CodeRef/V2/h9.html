<base>

Specifies the base URL/target for all relative URLs in a document.

Syntax

  • <base href="URL Goes Here">

Example

  • <head>
    <base href="https://www.w3schools.com" target="_blank">
    </head>

Browser Support

  • Chrome: Supported
  • Internet Explorer: Supported
  • Firefox: Supported
  • Safari: Supported
  • Opera: Supported
