<caption>

Defines a table caption.

Syntax

  • <caption>Content Goes Here</caption>

Example

  • <table>
    <caption>Caption</caption>
    <tr>
    <th>Caption 1</th>
    <th>Caption 2</th>
    </tr>
    <tr>
    <th>Caption Data 1</th>
    <th>Caption Data 2</th>
    </tr>
    </table>

Browser Support

  • Chrome: Supported
  • Internet Explorer: Supported
  • Firefox: Supported
  • Safari: Supported
  • Opera: Supported
